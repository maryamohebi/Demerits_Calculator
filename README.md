# Demerits Points Calculator 

A Flask web app that calculates demerits points on driving speed and speed limit. 

## Requirements
- Python 3.12
- Flask

# Tech Stack
- Python (Flask)
- HTML5
- CSS 
- JavaScript

# Features 
- Speed comparison logic
- Holiday period handling
- Flash message feedback

# How to run
macOS (Homebrew Python)
```bash
python3 flask_app.py
```
 Windows / Linux
```bash
python flask_app.py
```
